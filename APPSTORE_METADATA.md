### 説明文（4000文字以内）
```
Memoriaは、完全オフラインで動くAIチャットアシスタントです。すべての処理がiPhoneの中だけで完結し、電波がなくても応答し、入力した言葉は1バイトも外に送られません。さらにセッションをまたぐ長期記憶で、あなたとの会話を覚えていてくれます。

飛行機の中で、英語のメニューを翻訳したい。山奥のキャンプ場で、明日の予定を整理したい。電波が届かない地下鉄で、ふと浮かんだアイデアを書き留めたい。そんなとき、Memoriaは答えてくれます。

【こんなときに使える】

海外旅行・出張：電波が不安定な空港・機内・地方都市でも通信不要で翻訳・文章作成・調べ物。ローミング料金もデータ通信量もゼロ。

地下鉄・新幹線・山間部：オフライン環境でも止まりません。

機密性の高い相談：仕事、健康、お金、家族の話。クラウドに送りたくない内容も端末内で完結するので安心。

日記・学習ログ：書きためた内容を覚えていてくれます。毎回ゼロから自己紹介し直す必要なし。

語学学習：日本語・英語・スペイン語の3言語で会話・翻訳・文法チェック。覚えた表現をメモリに保存していつでも参照。

【Memoriaの特徴】

完全オフライン動作
すべての処理がiPhone内で完結。Gemma 3 1B（Google DeepMind）をローカル実行し、初回のみWi-Fiで約600MBダウンロードしたあとは機内モードでも完全動作。サーバー混雑もAPIエラーもありません。クラウド型AIは外部サーバー必須ですが、Memoriaは違います。海外の機内、山奥、地下鉄、災害時——通信が落ちる場所こそ本領。あなたの言葉も声も1バイトも外に出ません。

セッションをまたぐ長期記憶
オフラインで動くAIアプリの多くは記憶を持たず、新しい会話のたびに自己紹介し直す必要があります。Memoriaは会話を自動要約してSQLiteに保存し、新しい会話に過去の文脈を自動注入。クラウド型AIの記憶機能は月額課金とサーバー保管ですが、Memoriaの記憶はすべて端末内、外部送信ゼロ、買い切りです。

3言語ネイティブの語学パートナー
UI・音声入力ともに日本語・英語・スペイン語に対応。日本語で打って英語で返してもらう、文法を日本語で解説してもらうなど、言語をまたいだ使い方が自然にできます。海外移住・留学・ビジネス渡航中の語学サポートに最適。

オフライン音声入力
オンデバイス音声認識で3言語対応。声も外に出ません。発音練習、運転中の音声メモ、外国語シャドーイングにも。

強力なスラッシュコマンド
/english /japanese /spanish で翻訳。/cal で英語文法チェック。/grammar で文法解説。/remember でグローバルメモリに保存、/memory で一覧表示。さらに /addcommand で「丁寧語に直して」「TOEFL風に書き換えて」など自分専用の処理を自然言語のまま登録可能。コード不要です。

テーマ別セッション管理
仕事・学習・日記・趣味など、独立した記憶を持つセッションを作成可能。検索・エクスポート対応。

毎日触りたくなるデザイン
Catppuccin Mocha/Latteテーマ、iMessageライクなチャットUI、心地よいハプティクス。

【プライバシーファースト】
位置情報を収集しません。トラッキング・広告なし。データはすべてiPhone内のSQLiteに保存。iCloudバックアップ対象外（任意）。

【動作環境】
iOS 17.0以降 / iPhone対応。初回のみAIモデルをWi-Fiダウンロード（約600MB）、以降は通信不要。

【対応言語】
UI・音声入力: 日本語・英語・スペイン語

あなたのAI、あなたの記憶、あなたのデバイスの中だけで。プライバシーを犠牲にしないAIアシスタント、Memoriaを今すぐ。

No signup, no subscription, zero external transmission.

📚 Session management
Separate sessions for work, study, journal, hobbies — each with its own independent memory. Search and export supported.

🎨 Design you'll want to use daily
Catppuccin Mocha/Latte themes, iMessage-style chat bubbles, satisfying haptics.
