# 🧠 OllamaMemoryChat

**会話を記憶するローカルAIチャットアプリ for Mac**

> あなたのデータは、あなたのMacの中だけに保存されます。外部サーバーへの送信は一切ありません。

---

## ✨ 主な機能

- **長期記憶**: 会話を自動で要約・保存。次回起動後も文脈を継続
- **スラッシュコマンド**: `/english`（英訳）`/cal`（校正）など便利なコマンド
- **カスタムコマンド**: UIから自然言語で自分だけのコマンドを追加（コーディング不要）
- **マルチエンジン**: Ollama（ローカルAI）またはOpenAI互換API（自分のAPIキー）に対応
- **完全プライベート**: データはローカルのSQLiteに保存。外部送信ゼロ

---

## 🚀 起動方法

### 前提条件

- macOS 12以降
- Python 3.10以降（`python3 --version` で確認）
- [Ollama](https://ollama.ai) がインストール済みであること

### Ollamaのセットアップ（まだの場合）

```bash
# Ollamaのインストール（Homebrewを使う場合）
brew install ollama

# モデルのダウンロード例
ollama pull gemma3:4b

# Ollamaを起動
ollama serve
```

### アプリの起動

```bash
# ① 起動スクリプトで起動（初回はライブラリを自動インストール）
./launch_mac.sh
```

または

```bash
# ② .appとしてビルドしてから起動
./build_app.sh
open dist/OllamaMemoryChat.app
```

## 📖 使い方

### 基本的なチャット

アプリを起動してメッセージを入力するだけです。
会話が一定回数を超えると、自動的に記憶が圧縮・保存されます。

### スラッシュコマンド

`/` を入力するとコマンド候補が自動表示されます。

| コマンド | 説明 |
|---------|------|
| `/english [文章]` | 日本語をネイティブな英語に翻訳 |
| `/cal [文章]` | 文章の文法・表現を校正（日英対応） |
| `/memory` | 保存されている記憶サマリーを表示 |
| `/clear` | 現在の会話セッションをリセット（記憶サマリーは保持） |
| `/help` | コマンド一覧を表示 |

### カスタムコマンドの追加

設定画面（⚙️ボタン）から追加できます。

1. コマンド名を入力（例: `summarize`）
2. 説明を自然言語で入力（例: `入力した文章を3行で要約して`）
3. 「追加する」ボタンをクリック

これだけで `/summarize` コマンドが使えるようになります。

### AIエンジンの切り替え

設定画面から以下を切り替えられます：
- **Ollama（ローカル）**: 自分のPCで動くAI。完全オフライン対応
- **OpenAI互換API**: OpenAI・Anthropic・その他のAPIキーを使用

---

## ⚙️ 設定ファイル（config.yaml）

```yaml
app:
  port: 8765                    # ローカルサーバーのポート

ai:
  engine: "ollama"              # "ollama" or "openai_compatible"
  ollama_endpoint: "http://localhost:11434"
  ollama_model: "gemma3:4b"     # 使用するモデル名

memory:
  compression_threshold: 10    # 何往復で記憶を圧縮するか
```

## 📁 データの保存場所

| データ | 場所 |
|--------|------|
| 会話履歴・記憶 | `data/chat_memory.db`（SQLite） |
| カスタムコマンド | `data/chat_memory.db` |
| AI・アプリ設定 | `config.yaml` |

**注意**: `config.yaml` にAPIキーを設定した場合、ファイルのアクセス権限を適切に設定することを推奨します。

## 🔒 プライバシーについて

このアプリは **Privacy by Design** の原則で設計されています。

- ✅ 全データはあなたのMac内にのみ保存
- ✅ 開発者のサーバーは存在しない（データ収集ゼロ）
- ✅ 外部ネットワーク通信はあなたが設定したAIエンジンのみ
- ✅ ソースコードを自分で確認・修正できるオープンソース
